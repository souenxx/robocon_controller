robocon_controller
====
Overview

EV3rtを操作できるプログラム
## Description
ETロボコンの走行体であるEv3rtをパソコンから操作できるプログラムになります。
pcとEV3rtをBluetooth接続した上で、操作用プログラムを送り、仮想シリアルポートで入力を渡します。
大学のイベント用に作成し、イベント本番はパソコンと市販のゲームコントローラーで繋いでラジコン風にしました。
